# To-Do List

A **smart, text-driven to-do list web application** that automatically:
- Splits multiple tasks from a single sentence
- Detects time phrases like **today, tomorrow, next week, next month**
- Categorizes tasks based on the calendar
- Allows **manual priority assignment**
- Sorts and highlights tasks by urgency
- Displays a **modern 3D professional UI**
